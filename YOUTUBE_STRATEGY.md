### Automation Workflow

#### Option 1: AI-Assisted (Recommended)
1. **Script Generation** (Use Claude/ChatGPT)
   - Input: Topic + key points
   - Output: 60-90 second script with hooks

2. **Voice Generation** (ElevenLabs)
   - Clone your voice
   - Generate audio from script
   - Sounds like you, scales infinitely

3. **Video Creation** (Descript / CapCut)
   - Stock footage (Pexels/Storyblocks)
   - Text overlays
   - B-roll of credit reports, calculators

4. **Thumbnail Creation** (Canva API)
   - Template with face + big text
   - "I was SHOCKED" / "This changed everything"
   - High contrast, readable at small sizes

5. **Scheduling** (YouTube API)
   - Upload daily at 8 AM EST
   - Auto-post with description + links
   - End screens point to masterclass

#### Option 2: Batch Recording (Lower Tech)
1. Record 7 videos every Sunday (2-3 hours)
2. Edit Monday-Wednesday
3. Schedule all 7 for the week
4. Repeat weekly
