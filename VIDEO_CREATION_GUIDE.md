## üé¨ Video Assembly (CapCut or Canva)

### Using CapCut (Free, Recommended):

1. **Import Images**
   - Download all 6 generated images from Krea
   - Import into CapCut

2. **Set Timing**
   - Scene 1: 3 seconds
   - Scene 2: 7 seconds
   - Scene 3: 20 seconds
   - Scene 4: 15 seconds
   - Scene 5: 10 seconds
   - Scene 6: 5 seconds
   - Total: 60 seconds

3. **Add Text Overlays**
   - Use bold, readable font (Montserrat or Arial Bold)
   - White text with black outline (for visibility)
   - Position in center or lower third
   - Animate text (pop in or slide in)

4. **Add Voiceover**
   - Import your recorded audio
   - Sync with scenes
   - Add background music (low volume, upbeat, royalty-free)

5. **Add Effects**
   - Subtle zoom on each scene (Ken Burns effect)
   - Transitions between scenes (simple cuts or fades)
   - Sound effects for emphasis (optional)

6. **Export Settings**
   - Format: MP4
   - Resolution: 1080x1920 (9:16 for Shorts/Reels)
   - Frame rate: 30fps
   - Quality: High

## üé® Krea Prompt Tips

### To Get Consistent Character:
Add this to every prompt:
```
Same cartoon elephant character throughout, consistent character design, friendly elephant with round glasses, gray skin, big ears, professional but approachable
```

### Style Modifiers That Work Well:
- "Pixar style character design"
- "Vibrant colors, 4k quality"
- "Professional lighting"
- "Educational infographic style"
- "Clean background, minimal distractions"

### If Character Looks Different:
Use Krea's "Style Reference" feature:
1. Generate Scene 1 first
2. Save that image
3. Use it as style reference for Scenes 2-6
4. This ensures consistency
